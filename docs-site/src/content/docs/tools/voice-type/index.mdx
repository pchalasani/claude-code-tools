`voice-type` turns speech into text in whatever app
has focus -- your editor, browser, terminal, or a
coding agent's prompt box. Everything runs on your
machine: no cloud, no accounts, no audio leaving
your Mac.

:::note[Installation]
```bash
# recommended: Apple-GPU engine (best accuracy AND speed)
uv tool install "claude-code-tools[voice-mlx]"
voice-type --engine parakeet-mlx --segmentation hold
```
See [Configuration & CLI](./configuration/) for all
engines and options, or run `voice-type --help`.
:::

## Quick start

<Steps>

1. Install with the engine of your choice (see
   [engines](./configuration/#engines)):

   ```bash
   uv tool install "claude-code-tools[voice-mlx]"
   ```

2. Grant your terminal **Microphone**,
   **Accessibility**, and **Input Monitoring**
   permissions (System Settings → Privacy &
   Security) when prompted.

3. Start dictating:

   ```bash
   voice-type --engine parakeet-mlx --segmentation hold
   ```

   Press `Ctrl+;`, speak (watch the waveform pill),
   press `Ctrl+;` again -- the whole take types at
   your cursor. `Esc` cancels a take.

4. Make it yours: `voice-type init` writes a fully
   commented config to
   `~/.config/voice-type/config.toml`; after that,
   plain `voice-type` starts with your settings.

</Steps>

## The three modes

| Mode | Starts | Behavior |
|------|--------|----------|
| `toggle` | paused | hotkey starts/stops; pairs with `--segmentation hold` for whole-take accuracy |
| `wake` | passive | "hey claude ..." activates; stop phrase or silence re-arms; hotkey works as override |
| `vad` | dictating | hands-free; each utterance types when you pause |

## Hotkey & voice vocabulary

| Action | Default | Configurable as |
|--------|---------|-----------------|
| Toggle recording | `Ctrl+;` | `hotkey` |
| Cancel recording (discard) | `Esc` (only while recording) | `cancel_hotkey` |
| Re-type last transcript | off | `paste_hotkey` |
| Record a chord | `voice-type hotkey` | -- |
| Submit (press Enter) | say "go" / "over" / "submit" | `submit_phrases` |
| Stop dictating | say "stop listening" | `stop_phrase` |
